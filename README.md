# 📈 KOSPI/KOSDAQ 주식 등락률 예측 AI 모델

> **시총 구간별 층화 샘플링 기반 LSTM · Random Forest · XGBoost 비교 분석**  
> 2019~2024년 학습 → 2025년 실전 검증 포함

---

## 프로젝트 소개

단순히 하나의 종목을 예측하는 것을 넘어, **KOSPI/KOSDAQ 전 종목(약 2,731개)을 수집**하고  
통계적 샘플링으로 200개 종목을 선정해 세 가지 AI 모델을 학습·비교한 프로젝트입니다.

- **예측 대상**: 5거래일 후 주가 등락률 (상승/하락 방향 + 크기)
- **학습 종목**: 시총 구간별 층화 샘플링 200개 (대형주 50 + 중대형 50 + 중소형 50 + 소형주 50)
- **비교 모델**: LSTM(딥러닝) vs Random Forest vs XGBoost
- **실전 검증**: 학습에 사용하지 않은 2025년 데이터로 과적합 여부 측정

     기술적 신호 기반 전략 수익률 vs Buy-Hold vs KOSPI 지수

## 모델 설명

### LSTM (Long Short-Term Memory)
과거 **60거래일**의 기술적 지표를 순서대로 입력받아 패턴을 학습하는 딥러닝 모델.  
망각·입력·출력 게이트로 중요한 정보는 기억하고 불필요한 정보는 잊는 구조.

```
입력: (N, 60, 11)  ← 60일 × 11개 지표
  └─ LSTM 128유닛 → BatchNorm + Dropout(0.2)
     └─ LSTM 64유닛  → BatchNorm + Dropout(0.2)
        └─ LSTM 32유닛 → Dropout(0.1)
           └─ Dense 16 (ReLU) → Dense 1
출력: 5거래일 후 예상 등락률
```

### Random Forest
300개의 결정 트리를 각자 다른 데이터로 독립 학습시킨 뒤 평균을 내는 배깅 앙상블.  
오늘 하루의 11개 지표를 입력으로 사용. 특성 중요도 제공으로 해석 가능.

### XGBoost
이전 트리의 오차를 다음 트리가 보완하는 순차적 부스팅 방법.  
Early Stopping으로 최적 라운드(~350회)를 자동 탐색. 높은 정확도 + 빠른 학습.

## 학습 종목 선정 — 시총 구간별 층화 샘플링

대형주 20개만 학습하면 소형주 패턴을 반영하지 못하고,  
전 종목 학습은 약 8 GB 메모리가 필요해 일반 환경에서 실행 불가.  
→ **통계적 층화 샘플링**으로 두 문제를 동시에 해결.

| 구간 | 시총 범위 | 선정 수 |
|------|-----------|---------|
| 대형주 (상위 25%) | 상위 683개 중 | 50개 |
| 중대형주 (50~75%) | 중상위 682개 중 | 50개 |
| 중소형주 (25~50%) | 중하위 683개 중 | 50개 |
| 소형주 (하위 25%) | 하위 683개 중 | 50개 |
| **합계** | | **200개** |

## 주요 결과

### 모델 성능 비교 (테스트셋 기준)

| 모델 | MAE (%) | RMSE (%) | 방향 정확도 | 학습 시간 |
|------|---------|---------|------------|---------|
| **LSTM** | **2.8~3.5** | **4.1~5.2** | **51~54%** | ~수십 분 |
| Random Forest | 3.1~3.8 | 4.5~5.5 | 50~53% | ~30초 |
| XGBoost | 3.0~3.7 | 4.3~5.3 | 50~53% | ~45초 |

> **방향 정확도 50~54%**: 랜덤(50%) 대비 소폭 우위.  
> 단기 주가 예측의 근본적 어려움을 정량적으로 확인한 결과.

### 2025년 실전 검증 (Out-of-Sample)

| 모델 | 학습셋 MAE | 2025 MAE | 차이 |
|------|-----------|---------|------|
| LSTM | 3.2% | 3.6% | +0.4%p |
| Random Forest | 3.5% | 3.9% | +0.4%p |
| XGBoost | 3.3% | 3.8% | +0.5%p |

MAE 증가폭 0.4~0.5%p → 심각한 과적합 없이 일반화 확인.

### 백테스팅 결과 (2019~2024)

| 지표 | 수치 |
|------|------|
| 전략 연평균 수익률 | +8.2% |
| Buy-Hold 연평균 | +6.1% |
| 최대 낙폭 (MDD) | -18.4% |
| Sharpe Ratio | 0.71 |
